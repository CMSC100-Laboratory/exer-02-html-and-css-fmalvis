## Steps on How to Create a GitHub Page
1. Go to the settings of your repository
2. Set repository to public
3. Look for 'Pages' under 'Code and automation' in the left sidebar
4. Under 'Build and deployment', choose the branch that you will be working on
5. Click 'Save'
6. A link will then be provided where you can view your site
